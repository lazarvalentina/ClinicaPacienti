Descriere

ClinicaPacienti este o aplicație dezvoltată în Java pentru gestionarea pacienților și a programărilor într-o clinică medicală. 
Aplicația permite administrarea datelor și organizarea acestora într-un mod eficient prin utilizarea programării orientate pe obiect și a unei baze de date.

Funcționalități:
-gestionarea pacienților
-gestionarea programărilor
-adăugarea, modificarea și ștergerea datelor
-afișarea informațiilor existente
-interfață grafică pentru utilizator
